## Setup
1. Install Node.js (LTS) https://nodejs.org/
2. Run node -v  in CMD
       npm -v  

3. Open the '.ENV' file provided in the files, make sure its .ENV and not ENV and copy the code below: **(READ NOTES IF YOU DONT HAVE ENVIROMENT INSTALLED)**
   OPENAI_API_KEY=KEYHERE
   PORT=3000
4. In the folder with the files open the CMD or PowerShell and Start the server with this   
   command:    `node server.js`
5. Open `chatbot_frontend.html` in browser and test.

## Notes
- If you cannot make ENV files, open CMD in the root directory and type: npm install dotenv
- Do NOT commit `.env`.
- to get your own API KEY visit OpenAI website and obtain it from there. or click here:
  https://platform.openai.com/api-keys
